Поддерживаемые криптокошельком
блокчейны и токены

BTC

Биткоин — первая цифровая валюта, чей оборот никак не
регулируется — ни официальными органами, ни платежными
системами. У биткоина есть своя публичная сеть — блокчейн. Это
своеобразная база данных, в которой хранятся данные обо всех
транзакциях, представленную в виде цепочек. Ознакомиться с этой
информацией могут все, но изменить и уничтожить ее невозможно.

ETN

БЭфир (Ethereum) — это не только криптовалюта, но и полноценная
сеть для создания децентрализованных онлайн-сервисов на основе
технологии блокчейн. Для Ethereum создана сеть ERC20 -
официальная технология по внесению предложений для улучшения
сети. По сути, это унифицированный стандарт по созданию токенов
в блокчейне эфира.

BTC

Tron - уникальная платформа, созданная исключительно для
создания развлекательного контента. Она позволяет разработчикам
создавать собственные приложения с помощью специальных
инструментов. В качестве платежного средства здесь используется
токен TRX. Он позволяет оплачивать товары и услуги, проводить
транзакции. Все токены TRX должны соответствовать единому
стандарту TRC20.

Binace Smart Chain

BSC - сеть, разработанная биржей Binance, работающая на
технологии блокчейн, сформированная специально для программ
на основе смарт-контрактов (алгоритмов для формирования,
управления и передачи информации). Обычно BSC функционирует
совместно с ВС. Первая сеть обеспечивает доступ к смарт
-контрактам, а вторая - большую пропускную способность.
